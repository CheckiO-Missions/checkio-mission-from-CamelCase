You mission is to convert some function name (string) from CamelCase ("MyFunctionName") to Python format ("my_function_name") where all chars are in lowercase and all words are concatenated with underscore sign "_".

Input: Function name as CamelCase string

Output: The same string, but in under_score

Example:

{% if interpreter.slug == "js-node" %} 
fromCamelCase("MyFunctionName") == "my_function_name"
fromCamelCase("IPhone") == "i_phone"
fromCamelCase("ThisFunctionIsEmpty") == "this_function_is_empty"
fromCamelCase("Name") == "name"
{% else %} 
from_camel_case("MyFunctionName") == "my_function_name"
from_camel_case("IPhone") == "i_phone"
from_camel_case("ThisFunctionIsEmpty") == "this_function_is_empty"
from_camel_case("Name") == "name"
{% endif %}

How it is used: To use name of the functions in that style which are used in the language (Python, JavaScript, etc.).

Precondition:
0 < len(string) <= 100
no numbers in the input
